Detailed Description

This class represents the actual CHIP-8 and emulates it.

This class is the actual CHIP-8 emulator. It holds holds the memory it needs and runs the emulation. This class does not render itself and it does not provide a clock. To run the emulation, one has to call the methods step() and clockTimers() appropriately.
